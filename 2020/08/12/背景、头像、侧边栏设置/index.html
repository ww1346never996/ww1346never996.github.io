发表于 2020-08-12 阅读次数： Valine：

设置网站上的杂七杂八的东西

运营一个个人博客，光有文章可不行，网页的美观也是很重要的。这篇文章主要记录一些杂七杂八的设置，帮助你美化自己的博客网站。

背景的设置

随便找张你喜欢的背景图，重命名为background.jpg 方便找，然后丢到themes -> next -> source -> images 上面不一定是next，是你用的主题

themes -> next -> source -> css -> _custom ，找到路径下的custom.styl文件，没有就自己建一个，在文件的最上方加上一代码 body { background:url(/images/background.jpg（这是你之前加的背景图片的名字）);} 就完事了。

头像的设置

找到你要做头像的文件，重命名为avatar.jpg或者avatar.gif 使用头像旋转需要用.gif，但是我觉得转一圈挺蠢的，
